Quickstart
==========

Install
-------

.. code-block:: bash

   pip install -e .

Run a tiny geometry demo
------------------------

.. code-block:: bash

   spectrax-gk cyclone-info
   spectrax-gk cyclone-kperp --kx0 0.0 --ky 0.3

This prints the Cyclone base case defaults and a quick k_perp^2 range along the
field line.
