Roadmap
=======

This rebuild follows a staged plan with publishable milestones.

Stage 0
-------
- Repo scaffold, docs, tests, and basis utilities.
- Analytic flux-tube geometry for the Cyclone base case.

Stage 1
-------
- Linear electrostatic gyrokinetic operator in Hermite-Laguerre basis.
- Cyclone base case linear benchmarks.

Stage 2
-------
- Nonlinear E x B term with de-aliasing.
- Drift-wave turbulence in a box.

Stage 3
-------
- Electromagnetic extension and multi-species.
- Improved collision operators and closure.

Stage 4
-------
- VMEC/DESC geometry adapters and stellarator benchmarks.
- Performance tuning and scaling studies.

Publication milestones
----------------------

Each stage will produce a small, publication-quality figure and a short
validation note to make the progression transparent and review-friendly.
